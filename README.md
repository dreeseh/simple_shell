# Simple Shell

## Written by Reese Hicks and Blake Stewart Aug 18 2020

| File             |  Purpose                                            |
--------------------------------------------------------------------------
| AUTHORS          |  File listing the authors of this simple shell      |

| main.c           | Performs all of the forking and pid functions       |
